# MulaStar Website

A detailed, SEO-optimized website for **MulaStar** – the world's leading micro-task earning platform. Optimized for Google ranking with the keyword "MulaStar".

## Features

- **SEO Optimized**: Meta tags, Open Graph, Twitter Cards, canonical URL, Schema.org structured data (Organization, FAQPage)
- **Responsive Design**: Mobile-first, works on all devices
- **Accessible**: Skip link, ARIA labels, semantic HTML
- **Fast**: Minimal dependencies, vanilla JS

## Structure

- `index.html` – Single-page site with all sections
- `styles.css` – Responsive styling
- `script.js` – Mobile nav, FAQ accordion, smooth scroll

## Running Locally

Open `index.html` in a browser, or use a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve
```

Then visit `http://localhost:8000` (or the port shown).
